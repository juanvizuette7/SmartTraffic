

| Acción                       | Comando                  |
| ---------------------------- | ------------------------ |
| Detener todo normal          | `docker compose down`    |
| Pausar sin borrar redes      | `docker compose stop`    |
| Reiniciar rápido             | `docker compose start`   |
| Apagar y borrar todo (reset) | `docker compose down -v` |
| Iniciar para que arranque     | `docker compose up --build` |
